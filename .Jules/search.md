## Schema.org Requirements

### Organization (in `head/custom.html`)
Required: `@type`, `name`, `url`, `logo`, `telephone`, `email`, `address`, `sameAs`, `areaServed`

### Service (in `services.html`)
Required: `@type: Service`, `name`, `description`, `provider`, `areaServed`
Optional: `offers` with `PriceSpecification`

### BlogPosting (auto-generated by `jekyll-seo-tag`)
Verify: `headline`, `datePublished`, `author`, `description`, `image`

### FAQPage (in `faq.html`)
Required: `@type: FAQPage`, `mainEntity` array of `Question` + `acceptedAnswer`

### LocalBusiness (in `head/custom.html`)
Required: `@type`, `name`, `address`, `telephone`, `openingHours`, `priceRange`

## Concrete Measurement Commands

```bash
# Find all JSON-LD blocks
grep -rn 'application/ld+json' _includes/ _layouts/ *.html --include='*.html'

# Validate JSON-LD syntax (after build)
cat _site/index.html | python3 -c "import sys,json; [json.loads(b) for b in sys.stdin.read().split('application/ld+json\">')[1:]]" 2>&1

# Check for posts missing description
for f in _posts/*.md; do grep -L '^description:' "$f"; done

# Check description lengths (target: 120-160 chars)
for f in _posts/*.md; do
  desc=$(grep '^description:' "$f" | sed 's/description: *"*//;s/"*$//')
  len=${#desc}
  if [ $len -lt 120 ] || [ $len -gt 160 ]; then
    echo "⚠️ $f: description is $len chars (target: 120-160)"
  fi
done

# Check title lengths (target: ≤ 60 chars)
for f in _posts/*.md; do
  title=$(grep '^title:' "$f" | sed 's/title: *"*//;s/"*$//')
  len=${#title}
  if [ $len -gt 60 ]; then
    echo "⚠️ $f: title is $len chars (target: ≤60)"
  fi
done

# Find posts missing concepts
for f in _posts/*.md; do grep -L 'concepts:' "$f"; done

# Find duplicate titles
grep -h '^title:' _posts/*.md | sort | uniq -d

# Check for orphan pages (no internal links pointing to them)
for page in contact faq services meet-maria service-areas blog; do
  count=$(grep -rn "/$page/" _includes/ _layouts/ _posts/ *.html --include='*.html' --include='*.md' | wc -l)
  echo "$page: $count internal links"
done
```
